Policy modelling

CitizenCode builds transparent, data-driven tools that let anyone explore how UK policy decisions ripple through real populations. Anchored to ONS data. No jargon, no paywalls.

What is CitizenCode?

A personal project building open-source policy simulation tools for the UK. All models use publicly available ONS, HMRC, and UKHLS data. The code is on GitHub — read it, fork it, improve it.
